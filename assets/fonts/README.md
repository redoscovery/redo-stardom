# Fonts

Font files (`.otf`, `.ttf`) are not tracked in git due to size. Download them manually:

## Required

**Noto Sans CJK TC Regular** (16MB)

```bash
curl -L -o assets/fonts/NotoSansCJKtc-Regular.otf \
  "https://github.com/notofonts/noto-cjk/raw/main/Sans/OTF/TraditionalChinese/NotoSansCJKtc-Regular.otf"
```
