# REDÓ Stardom

以經典 DOS 遊戲《明星志願》系列為靈感的現代復刻版明星養成模擬遊戲。

本專案為 [REDÓSCOVERY](https://github.com/redoscovery) 系列作品之一。

## 遊戲概念

玩家扮演演藝經紀公司的經紀人，在架空世界中發掘、培養藝人，目標是在三年內讓旗下藝人登上巔峰、拿下三大獎項。

- **經紀人模式**為主要玩法，未來將支援藝人模式
- 融合傳統演藝圈與現代娛樂生態的元素
- 資料驅動設計，支援未來的 mod 擴充

## 技術棧

- **Rust** + **Bevy**
- 桌面原生應用
- PC-98 風格精細像素美術

## 狀態

## 聲明

本專案不使用原作的美術素材與遊戲內容，旨在以全新原創的方式復刻當年的遊戲體驗。
